# Quantitative Capstone Project

Applied econometrics and causal inference projects completed during my Master of International Affairs (International Economics) at UC San Diego's School of Global Policy and Strategy.

## Files

### `Final_project_capstone.R`
**Do State Earned Income Tax Credits Reduce Poverty? Evidence from a Staggered Difference-in-Differences Analysis of United States Counties**

Capstone research project evaluating the causal impact of state EITC adoption on county-level poverty rates. Constructed a 27-year panel dataset (1997–2023) covering 3,022 U.S. counties using Census Bureau SAIPE data. Applies the Callaway and Sant'Anna (2021) staggered difference-in-differences estimator, including pre-trend testing, parallel trends validation, and multiple robustness checks.

**Methods:** Staggered DiD, panel data construction, pre-trend testing, robustness checks
**Packages:** `did`, `fixest`, `ggplot2`, `tidyverse`

### `Final_project_analysis.R`
Supporting analysis and data visualization script for the capstone project above.

### `Ali_DFE_HW1_R_Code.R`
**Designing Field Experiments — Homework 1: Kenya Enterprise Promotion RCT**

Applied problem set analyzing a randomized controlled trial evaluating an enterprise promotion program in Kenya. Covers instrumental variables estimation, matching methods for comparison group selection, Lee Bounds for sample selection bias, and cost-benefit analysis of program impacts.

**Methods:** RCT analysis, IV/2SLS, matching methods, Lee Bounds, cost-benefit analysis
**Packages:** `AER`, `MatchIt`, `tidyverse`
